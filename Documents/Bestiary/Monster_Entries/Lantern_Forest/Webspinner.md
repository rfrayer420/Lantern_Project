# Webspinner

## Classification
Giant Arachnid

## Threat Level
III

## Habitat
High branches and webbed canopies

## Description
A giant spider that hangs lantern-shaped cocoons from trees.

## Statistics
- HP: 170
- Armor: 3
- Speed: Fast climb
- Damage: 2d6 bite plus venom

## Abilities
- Lantern Cocoon
- Web Trap
- Venom Bite

## Weaknesses
- Fire
- cutting anchor webs

## Loot
- Silk Strand
- Venom Sac

## Special Dice Rolls
Failed Dexterity roll means restrained in glowing webbing.

## Encounter Notes
Use for vertical forest encounters.
