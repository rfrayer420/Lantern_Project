# The Forgotten

## Classification
Lost Soul

## Threat Level
II

## Habitat
Dead-end corridors

## Description
Shadowy figures whose names and memories have been erased.

## Statistics
- HP: 80
- Armor: 1
- Speed: Slow
- Damage: 1d8 fear

## Abilities
- Erase Memory
- Mimic Voices
- Drain Hope

## Weaknesses
- Lantern light reveals their names

## Loot
- Memory Shards

## Special Dice Rolls
Critical failure causes loss of one important memory for 24 hours.

## Encounter Notes
Use as sorrowful threats rather than simple enemies.
