# The Hunger

## Classification
Apex Predator

## Threat Level
V

## Habitat
Anywhere memory weakens

## Description
A colossal beast that consumes memories, regrets, and souls.

## Statistics
- HP: 600
- Armor: 7
- Speed: Fast
- Damage: 4d10 plus memory drain

## Abilities
- Consume Memories
- Regeneration
- Reality Distortion
- Fear Aura

## Weaknesses
- Cannot feed on someone who accepts all regrets

## Loot
- Fragment of Memory
- Hunger Tooth

## Special Dice Rolls
Critical hit removes a skill until restored.

## Encounter Notes
Use sparingly as an apex nightmare.
