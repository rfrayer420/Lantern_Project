# Lantern Configuration Lore

## Artifact Transformation

Dormant State:
- Name: Lament Configuration
- Form: Sealed puzzle cube
- Status: Dormant / inactive
- Function: Conceals the Lantern Configuration within

Solved State:
- Name: Lantern Configuration
- Form: Active lantern configuration
- Status: Solved / active
- Function: Opens a path into the Labyrinth and summons The Forager

## Canon Rule

The Lament Configuration and the Lantern Configuration are the same artifact in different states.

Transformation sequence:

Lament Configuration (Dormant)
↓
Puzzle Solved
↓
Lantern Configuration (Active)
↓
Path Opens
↓
The Forager Arrives
